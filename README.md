# TEI-docker-runner
Effortless `Text Embeddings Inference(TEI)` Model deployment in Docker. Run and manage models with a single command.

## How to use 

#### dependency


* docker installed
* nvidia docker toolkit installed

### 1. clone repo

```
git clone x
cd x
```

### 2. configuration

### 2.1 cp `models.json`

```
cp models.json.example models.json
```

### 3. RUN

```
chmod +x run.sh
./run.sh
```
